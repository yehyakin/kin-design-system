# Long-Task Loaders

Mathematical curve animation is an exceptional progress treatment, not the default spinner.

## Candidate

[Math Curve Loaders](https://github.com/Paidax01/math-curve-loaders) may inspire a small branded indicator for genuinely long, multi-stage tasks.

## Eligibility

Use only when all are true:

- Expected duration is roughly 800ms or longer.
- The operation has a real stage, progress value or processed-item count.
- The user benefits from knowing the system is still working.
- A skeleton or inline pending state would not explain the operation better.

Examples:

- Global node detection.
- Domain-history analysis.
- Risk-report generation.
- Large message import.
- Relationship calculation.
- Large commerce catalog processing.

## Required content

```text
正在检测东京节点
18 / 32
```

Where supported, expose cancel, run in background or notify-on-completion actions.

## Visual contract

- Use one simple curve.
- 16–24px for inline use.
- Single semantic color.
- No particles, glow, long trail or continuous attention-seeking rotation.
- Do not replace accurate progress with animation.
- Reduced motion uses static progress or a conventional progress bar.

## Forbidden

- Initial page load.
- Search debounce.
- Route transition.
- Ordinary button submit.
- Image loading.
- Decorative empty states.

## Adoption decision

Do not copy source into KIN until license and maintenance are reviewed. A consuming product should prefer a small original implementation if the required behavior is trivial.
